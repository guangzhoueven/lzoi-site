【普及】统计稳定子数组的数目

给你一个长度为 nnn 的正整数数组 aaa

如果 aaa 的一个 连续子数组 中 没有逆序对 ，即不存在满足 i<ji < ji<j 且 a[i]>a[j]a[i] > a[j]a[i]>a[j] 的下标对，则该子数组被称为 稳定 子数组

现在有 qqq 个询问，每个询问需要统计完全包含在 a[li,⋯ri]a[l_i, \cdots r_i]a[li​,⋯ri​] 内的 稳定子数组 的数量

输入格式

第一行包含 222 个正整数 n,q(1≤n,q≤105)n, q(1 \le n, q \le 10^5)n,q(1≤n,q≤105)，表示数组 aaa 的长度和询问个数

第二行包含 nnn 个正整数，表示数组 aaa

接下来 qqq 行，每行包含 222 个数 li,ri(0≤li,ri≤n−1)l_i, r_i(0 \le l_i, r_i \le n - 1)li​,ri​(0≤li​,ri​≤n−1)

输出格式

输出 111 行包含 qqq 个数，表示每个询问的答案

3 3
3 1 2
0 1
1 2
0 2

2 3 4

2 2
2 2
0 1
0 0

3 1

提示

【样例 1 解释】

  • 对于询问 [li,ri]=[0,1][l_i, r_i] = [0, 1][li​,ri​]=[0,1]，子数组为 [3,1][3, 1][3,1]
    • 稳定子数组包括 [3][3][3] 和 [1][1][1]，稳定子数组的总数为 222
    • 子数组 [3,1][3, 1][3,1] 包含逆序对
  • 对于询问 [li,ri]=[1,2][l_i, r_i] = [1, 2][li​,ri​]=[1,2]，子数组为 [1,2][1, 2][1,2]
    • 稳定子数组包括 [1],[2],[1,2][1], [2], [1, 2][1],[2],[1,2]，稳定子数组的总数为 333
  • 对于询问 [li,ri]=[0,2][l_i, r_i] = [0, 2][li​,ri​]=[0,2]，子数组为 [3,1,2][3, 1, 2][3,1,2]
    • 稳定子数组包括 [3],[1],[2],[1,2][3], [1], [2], [1, 2][3],[1],[2],[1,2]，稳定子数组的总数为 444

【样例 2 解释】

  • 对于询问 [li,ri]=[0,1][l_i, r_i] = [0, 1][li​,ri​]=[0,1]，子数组为 [2,2][2, 2][2,2]
    • 稳定子数组包括 [2],[2],[2,2][2], [2], [2, 2][2],[2],[2,2]，稳定子数组的总数为 333
  • 对于询问 [li,ri]=[0,0][l_i, r_i] = [0, 0][li​,ri​]=[0,0]，子数组为 [2][2][2]
    • 稳定子数组包括 [2][2][2]，稳定子数组的总数为 111

【数据范围】

  • 1≤n,q≤1051 \le n, q \le 10^51≤n,q≤105
  • 1≤a[i]≤1051 \le a[i] \le 10^51≤a[i]≤105
  • 0≤li,ri≤n−10 \le l_i, r_i \le n - 10≤li​,ri​≤n−1
