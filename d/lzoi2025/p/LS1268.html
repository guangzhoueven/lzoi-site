【提高】异或序列

[提高] 异或序列

有一个长度为 nnn 的序列 aaa ，序列中的每个值在 [0,1024][0, 1024][0,1024] 之间。

请你求出这个序列有多少对连续子序列 (A,B)(A, B)(A,B) ，满足 AAA 在 BBB 之前，且 A,BA, BA,B 中所有元素的异或和为 mmm。

简单来说，你需要求出有多少个四元组 (l1,r1,l2,r2)(l_1, r_1, l_2, r_2)(l1​,r1​,l2​,r2​) ，满足 l1≤r1<l2≤r2l_1 \leq r1 < l_2 \leq r2l1​≤r1<l2​≤r2，且

$$(\bigoplus^{r_1}_{i=l_1}a_i)\bigoplus(\bigoplus^{r_2}_{i=l_2}a_i)=m$$

⨁\bigoplus⨁ 表示异或，即：

  • 0⊕0=00\oplus0=00⊕0=0
  • 0⊕1=10\oplus1=10⊕1=1
  • 1⊕0=11\oplus0=11⊕0=1
  • 1⊕1=01\oplus1=01⊕1=0

输入格式

第一行两个整数 n,mn, mn,m ，表示数组长度，异或和。

第二行 nnn 个整数，表示数组 aaa 。

输出格式

一行一个整数，表示答案。

保证答案不超过 long long 表示范围。

【样例 1 解释】

  • 以下是 333 种方案
  • $A = \lbrace0\rbrace, B = \lbrace2\rbrace, 0\oplus2=2$
  • $A = \lbrace1\rbrace, B = \lbrace3\rbrace, 1\oplus3=2$
  • $A = \lbrace0, 1\rbrace, B = \lbrace3\rbrace, 0\oplus1\oplus3=2$
