【提高】连续的自然数

题目描述

东东又在研究数列，他开始给数列命名。

他从 000 开始，看数列中是否包含 0, 1, 2, 3,⋯0,\ 1,\ 2,\ 3, \cdots0, 1, 2, 3,⋯

当数列不包含 xxx，就停止，这个数列就称为 数列 x。

  • 如数列 [2,9,1,0][2,9,1,0][2,9,1,0] 这个数列的名字就是 数列 3
  • 如数列 [3,1,0,2,6][3,1,0,2,6][3,1,0,2,6] 这个数列的名字就是 数列 4，因为里面包含了 0,1,2,30,1,2,30,1,2,3 但是没有 444。

给你一个长为 nnn 的数列 a1,a2,...,ana_1,a_2,...,a_na1​,a2​,...,an​。

有 mmm 次操作，每次操作让所有的 aia_iai​ 变成 ai+ia_i+iai​+i.

  • 如数列 [3,1,0,2,6][3,1,0,2,6][3,1,0,2,6]
  • 进行一次操作后变成 [3+1,1+2,0+3,2+4,6+5][3+1,1+2,0+3,2+4,6+5][3+1,1+2,0+3,2+4,6+5] 也就是 [4,3,3,6,11][4,3,3,6,11][4,3,3,6,11]

输出每次操作后数列的名字。

输入格式

第一行两个正整数 nnn 和 mmm。

接下来 nnn 个整数，表示数列。

输出格式

mmm行，每行 1 个整数，表示每次操作后数列的名字

3 3
-1 -1 -9

2 
0 
1

数据范围与提示

【样例 1 解释】

  • 进行第一次操作后，数列变成 [0,1,−6][0,1,-6][0,1,−6] 名字为 222
  • 进行第二次操作后，数列变成 [1,3,−3][1,3,-3][1,3,−3] 名字为 000
  • 进行第三次操作后，数列变成 [2,5,0][2,5,0][2,5,0] 名字为 111

【数据范围】

  • 40%:n,m≤10040\%: n,m \le 10040%:n,m≤100
  • $100\%: 1\le n,m \le 2 \times 10^5, -10^9\le a_i \le 10^9$
