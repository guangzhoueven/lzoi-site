【入门】逛超市_简单

题目描述

简单版和困难版不同是：简单版本中折扣券和立减券的数量均为 1，困难版本中折扣券和立减券的数量为给定值。

小明在逛超市。

他总共买了 nnn 件商品，第 iii 种商品的价格为 pip_ipi​

超市有下面的打折政策：

  • 每名顾客有 111 张折扣券，可以让一件商品的价格打折（如果此商品原价为 pip_ipi​，那么使用此优惠券后，价格变为 pi⋅x%p_i \cdot x\%pi​⋅x%
  • 每名顾客有 111 张立减券，可以让一件商品的价格减小 yyy（如果此商品原价小于 yyy，那么可以花费 000 买下）。
  • 每个商品最多使用 111 张优惠券。

请求出小明可能付出的最小的花费。

输入格式

第一行包含一个整数 T(1≤T≤105)T(1 \le T \le 10^5)T(1≤T≤105)，表示测试用例的组数。

对于每组测试用例：

第一行包含三个整数 $n(1 \le n \le 2 \times 10^5), x(1 \le x \le 99), y(1 \le y \le 10^4)$

第二行包含 nnn 个整数 p1,p2,...,pn(1≤pi≤104)p_1, p_2, ..., p_n(1 \le p_i \le 10^4)p1​,p2​,...,pn​(1≤pi​≤104)，表示商品的价格

保证对于所有的测试用例，nnn 的总和不超过 2⋅1052 \cdot 10^52⋅105

输出格式

对于每组测试用例： 仅输出一行，包含一个实数，表示答案。如果你的答案和标准答案的绝对误差或相对误差不超过 10−410^{-4}10−4，则你的答案会被视为正确。
