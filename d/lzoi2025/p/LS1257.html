【入门】2数之差

给定一个长度为 nnn 的 排序好 的数组 a 和整数 mmm，请你求出这样的数对 i,ji, ji,j 的个数:

  • 0≤i,j≤n−10 \le i, j \le n - 10≤i,j≤n−1
  • i<ji < ji<j
  • a[j]−a[i]≥ma[j] - a[i] \ge ma[j]−a[i]≥m

输入格式

第一行包含 222 个整数 n,m(2≤n≤105,−109≤m≤109)n, m(2 \le n \le 10^5, -10^9 \le m \le 10^9)n,m(2≤n≤105,−109≤m≤109)

第二行包含 nnn 个整数，表示数组 a(−109≤ai≤109)a(-10^9 \le a_i \le 10^9)a(−109≤ai​≤109)

数组 aaa 保证 按从小到大排序

输出格式

输出 111 行包含 111 个数，表示答案

4 5
2 5 7 11

3

提示

【样例 1 解释】

  • 7−2=57 - 2 = 57−2=5
  • 11−2=911 - 2 = 911−2=9
  • 11−5=611 - 5 = 611−5=6
  • 所以答案是 333
