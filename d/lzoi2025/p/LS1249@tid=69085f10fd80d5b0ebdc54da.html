【普及】异或和

给定长为 nnn 的数列 aaa，计算和

$$\displaystyle (\sum_{i=1}^{n} \sum_{j=i}^{n} a_i \text{ xor } a_j) \bmod 998244353$$

这里  xor \text{ xor } xor  表示按位异或，在 C/C++ 中表示为 ^

什么是 xor 运算？
  • 0 xor 0=00 \text{ xor } 0 = 00 xor 0=0
  • 0 xor 1=10 \text{ xor } 1 = 10 xor 1=1
  • 1 xor 0=11 \text{ xor } 0 = 11 xor 0=1
  • 1 xor 1=01 \text{ xor } 1 = 01 xor 1=0
  • 也就是相同为 0，不同为 1

输入格式

第一行包含 111 个正整数 n(2≤n≤105)n(2 \le n \le 10^5)n(2≤n≤105)

第二行包含 nnn 个整数表示 ai(0≤ai≤231−1)a_i(0 \le a_i \le 2^{31}-1)ai​(0≤ai​≤231−1)
