【入门】数据流中的众数

题目描述

给你 1 个空数组 aaa，现在要依次插入 nnn 个数到数组 aaa 里面，请在每次插入后输出，当前数组中众数的个数，以及每个众数出现的次数

输入格式

第 1 行包含 1 个整数正 n(1≤n≤105)n(1 \le n \le 10^5)n(1≤n≤105), 表示操作的个数

接下来 1 行包含 nnn 个正整数，用空格分开

输出格式

输出 nnn 行，每行 2 个整数用空格分开，分别表示每次插入后众数的个数，和每个众数出现的次数

7
1 2 3 1 2 3 1

1 1
2 1
3 1
1 2
2 2
3 2
1 3

提示

  • 样例解释
  • 当 a = [1] 时，众数是 [1], 出现的次数是 1
  • 当 a = [1,2] 时，众数是 [1,2], 出现的次数是 1
  • 当 a = [1,2,3] 时，众数是 [1,2,3], 出现的次数是 1
  • 当 a = [1,2,3,1] 时，众数是 [1], 出现的次数是 2
  • 当 a = [1,2,3,1,2] 时，众数是 [1,2], 出现的次数是 2
  • 当 a = [1,2,3,1,2,3] 时，众数是 [1,2,3], 出现的次数是 2
  • 当 a = [1,2,3,1,2,3,1] 时，众数是 [1], 出现的次数是 3
  • 考虑用 map<int, int> 实时存储每个数出现的次数
map<int, int> q;
for (int i = 0; i < n; i++) {
    cin >> x;
    // 当 map 中不存在 x 时, q[x] 默认是 0
    q[x]++;
}
