【入门】上网统计

题目描述

在一个网络系统中有 nnn 个用户，mmm 次上网记录。

每个用户可以自己注册一个用户名，每个用户名是一个只含小写字母且长度小于 1000 的字符串。

每个上网的账号每次上网都会浏览网页，网页名是以一个只含小写字母且长度小于 1000 的字符串。

每次上网日志里都会有记录，现在请统计每个用户每次浏览了多少个网页。

输入格式

第 1 行包含两个用 1 个空格隔开的正整数 n(1≤n≤1000)n(1 \le n \le 1000)n(1≤n≤1000) 和 m(1≤m≤5000)m(1 \le m \le 5000)m(1≤m≤5000)

第 222 到 m+1m + 1m+1 行，每行包含 2 个用 1 个空格隔开的字符串，分别表示用户名和浏览的网页名。

输出格式

共 nnn 行，每行的第一个字符串是用户名，接下来的若干字符串是这个用户依次浏览的网页名（之间用一个空格隔开）。按照用户名出现的次序排序输出。

5 7
goodstudyer bookshopa
likespacea spaceweb
goodstudyer bookshopb
likespaceb spaceweb
likespacec spaceweb
likespacea juiceshop
gameplayer gameweb

goodstudyer bookshopa bookshopb
likespacea spaceweb juiceshop
likespaceb spaceweb
likespacec spaceweb
gameplayer gameweb

提示

  • 考虑用 map<string, vector> 来存储数据
map<string, vector<string>> q;  // 存储每个人看的网页
for (int i = 0; i < m; i++) {
    cin >> name >> web;
    q[name].push_back(web);
}
