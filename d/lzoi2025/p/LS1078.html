【入门】最大化平均值

题目描述

有 nnn 个物品的体积和价值分别是 viv_ivi​ 和 wiw_iwi​，从中选出 kkk 个物品使得单位体积的价值最大

输入格式

第一行包含 222 个正整数 n,kn, kn,k。

第二行包含 nnn 个用空格分割的整数，表示物品的价值 wiw_iwi​

第二行包含 nnn 个用空格分割的整数，表示物品的体积 viv_ivi​

输出格式

输出一个浮点数，表示选中 kkk 个物品的最大单位价值

注意，结果保留 222 位小数。

3 2
2 3 1
2 5 2

0.75

提示

【样例解释 1】

  • 选择 1 号物品和 3 号物品
  • 总价值: 2+1=32 + 1 = 32+1=3，总体积: 2+2=42 + 2 = 42+2=4
  • 所以答案是: 4/3=0.754 / 3 = 0.754/3=0.75
  • 贪心地选择单位价值最高的前 kkk 个物品是否正确？
