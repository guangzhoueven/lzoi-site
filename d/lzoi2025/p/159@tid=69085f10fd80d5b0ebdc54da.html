【入门】子数组之和

给你一个长度为 nnn 的数组 aaa，定义：

f(l,r)=∑i=lraif(l, r) = \sum_{i = l}^{r}a_i f(l,r)=i=l∑r​ai​

请你计算下面式子的结果：

∑i=1n∑j=inf(i,j)\sum_{i=1}^{n}\sum_{j=i}^{n}f(i,j) i=1∑n​j=i∑n​f(i,j)

简单来说：

  • f(l,r)f(l, r)f(l,r)：表示 子数组 [al,al+1,⋯ ,ar][a_l, a_{l+1}, \cdots, a_{r}][al​,al+1​,⋯,ar​] 的和
  • 求所有子数组的 f(l,r)f(l, r)f(l,r)，再相加，求最后的总和

由于答案可能很大，你只需要输出答案对 998244353 取模后的结果

输入格式

第一行包含一个整数 nnn 代表数组长度；

第二行包含 nnn 个整数表示数组 aaa

输出格式

对于每组数据输出一行，包含答案

3
1 2 3

20

【样例 1 解释】

  • f([1])=1f([1]) = 1f([1])=1
  • f([1,2])=3f([1, 2]) = 3f([1,2])=3
  • f([1,2,3])=6f([1, 2, 3]) = 6f([1,2,3])=6
  • f([2])=2f([2]) = 2f([2])=2
  • f([2,3])=5f([2, 3]) = 5f([2,3])=5
  • f([3])=3f([3]) = 3f([3])=3
  • 结果为：1+3+6+2+5+3=201+3+6+2+5+3=201+3+6+2+5+3=20
