【入门】二维vector

题目描述

给你 nnn 个 vector 数组，分别是 ai(i=0,1,2...,n−1)a_i(i=0,1,2...,n-1)ai​(i=0,1,2...,n−1)，初始都为空数组，现在有如下 3 种操作

  • 0 t x， 表示将 x 添加到 ata_tat​ 的末尾
  • 1 t，表示按需输出 ata_tat​ 中的所有元素
  • 2 t，表示将 ata_tat​ 清空

输入格式

第 1 行包含 2 个整数正 n(1≤n≤1000),q(1≤q≤5∗105)n(1 \le n \le 1000), q(1 \le q \le 5*10^5)n(1≤n≤1000),q(1≤q≤5∗105)，表示数组的个数和操作的个数

  • 当 c=2c = 2c=2 时，再输入 1 个数 ttt，表示清空数组 ata_tat​

  • 样例解释
  • 0 0 1，[c = 0, t = 0, x = 1], 代表把 x = 1 放到 a[t] 末尾，a[0] = [1]
  • 0 0 2，执行后 a[0] = [1, 2]
  • 0 0 3，执行后 a[0] = [1, 2, 3]

  • 二维 vector 的语法如下
// 申明 n 个 1维 vector<int>
vector<vector<int>> a(n);
for (int i = 0; i < q; i++) {
    cin >> t >> x;
    a[t].push_back(x);
}

// 清空 vector
a[t].clear();

// 遍历 vector<int> 有 3 中方法
// 1、获取 size
for (int i = 0; i < a[t].size(); i++) cout << a[t][i] << ' ';

// 2、使用迭代器
for (int i = a[t].begin(); i != a[t].end(); i++) cout << *i << ' ';

// 3、最简单方法
for (int i : a[t]) cout << x << ' ';

// 可以用 auto 代替 int
for (auto i : a[t]) cout << x << ' ';
