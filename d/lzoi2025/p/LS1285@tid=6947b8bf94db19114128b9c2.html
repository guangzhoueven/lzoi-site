【入门】构造单调数组

题目描述

样例和数据有更新！！

给你 333 个正整数 a,b,ca, b, ca,b,c，一次操作可以将 a,b,ca, b, ca,b,c 中任意一个数减 1（不能减到 000），请问最少需要几次操作，可以使得 a<b<ca < b < ca<b<c；如果不能做到，请输出 -1

输入格式

第一行包含一个整数 TTT，表示数据组数；

每组数据包含 111 行 333 个整数 a,b,ca, b, ca,b,c

输出格式

对于每组数据输出一行包含答案

5
24 40 24
10 8 5
5 4 2
2 2 4
1 1 3

19
11
-1
1
-1

提示

【样例 1 解释】

  • 可以将 24,40,2424, 40, 2424,40,24 变为 22,23,2422, 23, 2422,23,24
  • 总的操作次数：$(24 - 22) + (40 - 23) + (24 - 24) = 2 + 17 + 0 = 19$

【样例 2 解释】

  • 可以将 10,8,510, 8, 510,8,5 变为 3,4,53, 4, 53,4,5
  • 总的操作次数：(10−3)+(8−4)+(5−5)=7+4+0=11(10 - 3) + (8 - 4) + (5 - 5) = 7 + 4 + 0 = 11(10−3)+(8−4)+(5−5)=7+4+0=11

【样例 4 解释】

  • 可以将 2,2,42, 2, 42,2,4 变为 1,2,41, 2, 41,2,4
  • 总的操作次数：(2−1)+(2−22)+(4−4)=1+0+0=19(2 - 1) + (2 - 22) + (4 - 4) = 1 + 0 + 0 = 19(2−1)+(2−22)+(4−4)=1+0+0=19

【数据范围】

对于所有测试数据，均有：

  • 1≤a,b,c≤1091 \le a, b, c \le 10^91≤a,b,c≤109
