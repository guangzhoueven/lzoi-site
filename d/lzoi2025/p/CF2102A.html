【入门】构造数组

题目描述

给定四个整数 nnn 、 mmm 、 ppp 和 qqq ，判断是否存在满足以下条件的整数数组 a1,a2,…,ana_1, a_2, \ldots, a_na1​,a2​,…,an​ (元素可能为负数)：

  • 数组中所有元素的和等于 mmm ：

    a1+a2+…+an=ma_1 + a_2 + \ldots + a_n = m a1​+a2​+…+an​=m

  • 每 ppp 个连续元素之和等于 qqq ：

    $$a_i + a_{i + 1} + \ldots + a_{i + p - 1} = q,\qquad\text{ for all }1\le i\le n-p+1$$

输入格式

输入

每个测试包含多个测试用例。第一行包含测试用例的数量 ttt ( 1≤t≤1041 \le t \le 10^41≤t≤104 )。测试用例说明如下。

每个测试用例的第一行也是唯一一行包含四个整数 nnn 、 mmm 、 ppp 和 qqq （ 1≤p≤n≤1001 \le p \le n \le 1001≤p≤n≤100 、 1≤q,m≤1001 \le q, m \le 1001≤q,m≤100 ）--分别是数组的长度、元素的总和、段的长度和段的总和。

输出格式

输出

对于每个测试用例，如果存在满足上述条件的数组，则输出 "YES"（不带引号），否则输出 "NO"（不带引号）。

【样例 1 解释】

在第一个测试用例中，满足条件的数组示例是 [1,0,1][1, 0, 1][1,0,1] 。这是因为

  • a1+a2+a3=1+0+1=2=ma_1+a_2+a_3 = 1+0+1 = 2 = ma1​+a2​+a3​=1+0+1=2=m
  • a1+a2=1+0=1=qa_1+a_2=1+0=1=qa1​+a2​=1+0=1=q
  • a2+a3=0+1=1=qa_2+a_3=0+1=1=qa2​+a3​=0+1=1=q

在第二个测试用例中，唯一满足条件的数组是 [1][1][1] 。

在第三个测试用例中，满足条件的数组是 [−2,5,−2,5,−2][-2, 5, -2, 5, -2][−2,5,−2,5,−2] 。

在第四个测试用例中，可以证明没有满足条件的数组。
