【普及】求和游戏

给你一个长度为 nnn 的数组 aaa 和 222 个整数 l,r(l≤r)l, r(l \le r)l,r(l≤r)，游戏规则如下：

  • 1、每一轮你可以从数组的左边开始，连续拿走若干个数
  • 2、如果拿走的这些数的和在 lll 到 rrr 之间，你可以获得 111 分，否则不能得分

你可以进行多轮游戏，直到数组为空；请问你最多可以获得多少分？

输入格式

第一行包含 111 个整数 TTT，表示数据组数

每组数据的第一行包含 333 个整数 n,l,r(1≤n≤105n, l, r(1 \le n \le 10^{5}n,l,r(1≤n≤105, 1≤l≤r≤109)1 \le l \le r \le 10^9)1≤l≤r≤109)

每组数据的第二行包含 nnn 个整数 a1,a2,…,an(1≤ai≤109)a_1, a_2, \ldots, a_n (1 \le a_i \le 10^9)a1​,a2​,…,an​(1≤ai​≤109)

保证所有数据的 nnn 之和不超过 2×1052 \times 10^52×105

【样例 1 解释】

在第一个测试案例中，你可以获得 333 分：

  • 在第一轮中，取牌值为 [2,1][2, 1][2,1]，它们的总和是 3(3≤3≤10)3(3 \le 3 \le 10)3(3≤3≤10)，你可以获得 111 分，数组会变成：[11,3,7][11,3,7][11,3,7]
  • 第二轮，取 [11][11][11]，无法得分，因为其和为 11>1011>1011>10，数组会变成：[3,7][3,7][3,7]
  • 第三轮，取 [3][3][3]，你可以获得 111 分，因为其和为 3≤3≤103\le 3 \le 103≤3≤10，数组会变成：[7][7][7]
  • 第四轮，取 [7][7][7]，你可以获得 111 分，因为其和为 3≤7≤103\le 7 \le 103≤7≤10，数组被取空
  • 所以你能获得 333 分
