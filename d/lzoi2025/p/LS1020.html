【入门】数组的无限操作

题目描述

有一个长为 nnn 的数组 aaa，数组的下标为 1,2,3...,n1,2,3...,n1,2,3...,n 。他可以进行任意多次如下这种操作：

对于两个正整数 iii 和 jjj, (1≤i≤n1 \le i \le n1≤i≤n)，如果此时数组满足 ai+aja_i+a_jai​+aj​ 是偶数，则交换 aia_iai​ 和 aja_jaj​ 的值。 现在我们想知道，他在任意次操作之后，能得到的字典序最小的数组是什么？

  • 什么是字典序？
    • 对于两个数组 aaa 和 bbb
    • 找到最小的 iii 使得 ai≠bia_i \neq b_iai​=bi​
    • 如果 ai<bia_i \lt b_iai​<bi​，就说 aaa 的字典序小于 bbb；
    • 如果 ai>bia_i \gt b_iai​>bi​，就说 aaa 的字典序小于 bbb；
    • 如果找不到这样的 i, 那么 aaa 和 bbb 中长度小的数组，字典序更小
  • 下面的例子都是 aaa 的字典序小于 bbb 的字典序
    • a=[1,2,3],b=[1,3,2]a = [1, 2, 3], b = [1, 3, 2]a=[1,2,3],b=[1,3,2]
    • a=[1,2,3],b=[1,2,3,4]a = [1, 2, 3], b = [1, 2, 3, 4]a=[1,2,3],b=[1,2,3,4]
