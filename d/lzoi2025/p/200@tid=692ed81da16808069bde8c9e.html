【普及】异或子数组

给你一个长度为 nnn 的整数数组 a[]a[]a[]，请你求出同时满足以下两个条件的 最长子数组 的长度 ：

  • 1、子数组的按位异或（XOR）为 000
  • 2、子数组包含的 偶数 和 奇数 数量相等。 如果不存在这样的子数组，则输出 000。

子数组 是数组中的一个 连续、非空 元素序列。

输入格式

第一行包含 111 个整数 TTT，表示数据组数

每组数据的第一行包含 111 个整数 n(1≤n≤105n(1 \le n \le 10^{5}n(1≤n≤105)$

每组数据的第二行包含 nnn 个整数 a1,a2,…,an(1≤ai≤109)a_1, a_2, \ldots, a_n (1 \le a_i \le 10^9)a1​,a2​,…,an​(1≤ai​≤109)

保证所有数据的 nnn 之和不超过 2×1052 \times 10^52×105

输出格式

对于每组数据输出 111 行包含 111 个数，表示 最长子数组 的长度

8

提示

【样例 1 解释】

  • 子数组 [1,3,2,0][1, 3, 2, 0][1,3,2,0] 的按位异或为 1 ^ 3 ^ 2 ^ 0 = 0
  • 且包含 2 个偶数和 2 个奇数

【样例 2 解释】

  • 整个数组的按位异或为 000
  • 且包含 444 个偶数和 444 个奇数

【样例 3 解释】 没有非空子数组同时满足两个条件

【数据范围】

  • 1≤T≤1041 \le T \le 10^{4}1≤T≤104
  • 1≤n≤1051 \le n \le 10^{5}1≤n≤105
  • 0≤ai≤1090 \le a_i \le 10^90≤ai​≤109
  • 保证同一组内的 nnn 之和不超过 2×1052 \times 10^52×105
