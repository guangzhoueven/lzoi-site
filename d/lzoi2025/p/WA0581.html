题目描述

输入一个只包含加法和减法的数学表达式，输出表达式的答案。

提示：

    // 每次遍历到 + (或者 -)
    // 就顺便把后面的数处理出来

输入格式

一行为一个不包含空格的数学表达式。

输出格式

输出表达式的答案。

1+2-3+4

4

1000000000-33333333+1000000000+3333

1966670000

1000000000+1000000000+1000000000

3000000000
