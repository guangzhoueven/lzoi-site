【入门】AB数对

[入门] 序列

给出一串正整数数列以及一个正整数 CCC，要求计算出所有满足 A−B=CA - B = CA−B=C 的数对（A， B）的个数

不同位置的数字一样的数对算不同的数对。

提示：

  • 注意观察数据范围，比如数组的长度，计算下时间复杂度，超过 O(108)O(10^8)O(108) 是会超时的
  • 注意观察数据范围，比如 1≤C<1091 \leq C < 10^91≤C<109
  • 想一想最近学过的工具，哪个可以用上，翻一翻这几天的课件，你是做过类似题目的
  • 想一想最后的结果会不会超过 int 的最大值

输入格式

输入共两行。

第一行，两个正整数 N,CN,CN,C。

第二行，NNN 个正整数，作为要求处理的那串数。

输出格式

一行，表示该串正整数中包含的满足 A−B=CA - B = CA−B=C 的数对（A， B）的个数。

4 1
1 1 2 3

3

提示

【样例 1 解释】

  • (1, 2), (1, 2), (2, 3) 是满足条件的 333 个数对

【数据范围】

  • 对于 20%20\%20% 的数据，1≤N≤20001 \leq N \leq 20001≤N≤2000。
  • 对于 100%100\%100% 的数据，1≤N≤2×1051 \leq N \leq 2 \times 10^51≤N≤2×105，0≤ai<1090 \leq a_i < 10^90≤ai​<109，1≤C<1091 \leq C < 10^91≤C<109。
