[海淀区小学组 2023] 赛车游戏

题目背景

2023 年海淀区中小学生信息学竞赛小学组复赛题目，数据为洛谷自造。

题目描述

陶陶和天天喜欢玩赛车游戏，在游戏中有一条直赛道长度为 l l l，陶陶的赛车在起点为 000 的位置，准备向终点行驶，天天的赛车在终点为 l l l 的位置，准备向起点行驶。赛车的初始速度都为 111，在赛道上有 n n n 个加速带，第 i i i 加速带的位置为 ai a_i ai​，当小车经过一个加速带时，它的速度就增加 111，请你帮忙计算出两车相遇时间。

输入格式

第一行仅有一个整数 T T T 表示测试数据的组数，每组测试数据的第一行包含两个整数 n n n 和 l l l，第二行包含 n n n 个整数 a1,a2,a3,…,an a_1, a_2, a_3, \ldots, a_n a1​,a2​,a3​,…,an​。

输出格式

共有 T T T 行，每行仅有一个数，依次对应每组测试数据的答案，表示两车相遇的时间。允许绝对误差、相对误差不超过 10−6 10^{-6} 10−6。
