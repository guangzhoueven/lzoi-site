[APIO2019] 路灯

题目描述

一辆自动驾驶的出租车正在 Innopolis 的街道上行驶。该街道上有 n+1n+1n+1 个停车站点，它们将街道划分成了 nnn 条路段。每一路段都拥有一个路灯。当第 iii 个路灯亮起，它将照亮连接第 iii 与第 i+1i+1i+1 个站点的路段。否则这条路段将是黑暗的。

安全起见，出租车只能在被照亮的路段上行驶。换言之，出租车能从站点 aaa 出发到达站点 b(a<b)b (a<b)b(a<b) 的条件是：连接站点 aaa 与 a+1a+1a+1，a+1a + 1a+1 与 a+2a+2a+2，……，b−1b-1b−1 与 bbb 的路段都被照亮。

在经过一些意外故障或修理之后，街道上的路灯可能是亮起的，也可能是熄灭的。

现在给定 000 时刻时，街道上路灯的初始状态。之后 1,2,…,q1,2,\ldots,q1,2,…,q 时刻，每时刻会发生下列两种事件之一：

  • toggle i\text{toggle} \ itoggle i：切换第 iii 个路灯的状态。具体地说，若路灯原来亮起，则现在将熄灭；若路灯原来熄灭，则现在将亮起。

  • query a b\text{query} \ a \ bquery a b：出租车部门的负责人想知道，从 000 时刻起到当前时刻，有多少个时刻满足：出租车能够从站点 aaa 出发到达站点 bbb。

请你帮助出租车部门的负责人回答他们的问题。

输入格式

第一行包含两个整数 nnn 和 qqq，表示路灯的数量与时刻数。

第二行包含一个字符串 sss 表示路灯的初始状态，sis_isi​ 为 111 表示第 iii 个路灯初始时亮起；sis_isi​ 为 000 表示第 iii 个路灯初始时熄灭。

接下来 qqq 行每行描述一个时刻的事件。第 iii 行描述时刻 iii 所发生的事件：

  • toggle i\text{toggle} \ itoggle i：该时刻切换了第 iii 个路灯的状态。

  • query a b\text{query} \ a \ bquery a b：计算从 000 时刻起到该时刻，共有多少个时刻满足：出租车能从站点 aaa 出发到达站点 bbb。

至少有一个时刻的事件是 query\text{query}query。
